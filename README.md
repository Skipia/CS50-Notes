## Introdução
Este repositório nasceu como um **hobby** e um desafio pessoal. Aqui, registro minha jornada através do **CS50x: Introduction to Computer Science** da Universidade de Harvard. 

O objetivo deste espaço é centralizar meus estudos sobre lógica de programação, algoritmos e estruturas de dados, enquanto aprendo na prática a dominar ferramentas como **Git**, **GitHub** e o fluxo de trabalho com o **Obsidian**.

## Habilidades em Desenvolvimento
Este projeto foca na transição do pensamento comum para o pensamento computacional. Algumas habilidades trabalhadas:
* **Lógica de Programação:** Entendimento de funções, loops e condicionais.
* **Gerenciamento de Memória:** Manipulação de ponteiros e alocação dinâmica em C.
* **Estruturas de Dados:** Arrays, Listas Ligadas, Árvores e Tabelas Hash.
* **Versionamento de Código:** Uso real de Git (Commits, Push, Branching e .gitignore).

---

## Estrutura do Repositório
As notas estão organizadas por semanas, seguindo o cronograma do curso:
* `Week 0 - Scratch`: Lógica visual e conceitos iniciais.
* `Week 1 - C`: Sintaxe básica e tipos de dados.
* `Week 2 - Arrays`: Organização de dados em sequência.
* *(Em progresso...)*
